Claude Code（クロード・コード）は、あなたの言葉で頼むだけで、調べる・書く・直す・動かすをAIが代わりに進めてくれる道具。むずかしい専門用語はナシ。MacとGoogle Chromeがあれば、0から始められます。

案内役はこの子、うごく君。つまずきやすいポイントを先回りして教えます。

うごく君より

はじめまして、うごく君です。「コードとか、ターミナルとか、自分には無理そう…」と身構えなくて大丈夫。この記事を上から順に進めるだけで、次のことができるようになります。

  • MacにClaude Codeを入れて、自分の名前でログインできる
  • AIに「〇〇して」と日本語で頼んで、作業を任せることができる
  • 料金プランの予算感がわかり、自分に合う一枚を選べる
  • 仕事にも遊びにも使える、安全な使い方の勘どころがわかる
30秒でわかる

そもそもClaude Codeって、何が違うの？

同じ「Claude」でも、入口が3つあります。スマホのアプリと同じで、得意分野と“できること”が違います。

💬

Claude ＝ 相談する・作る

ブラウザやアプリでお願いすると、文章・要約・翻訳・アイデアが返ってくる。画面の中で“会話”する使い方。まずはここから。

🖥️

Claude Code ＝ 代わりに作業する

あなたのMacの中で、ファイルを読み・書き・整理し・実際に動かしてくれる“作業してくれるAI”。指示するだけで手が動く、今回の主役。

🧩

Claude Desktop ＝ 画面で操作する

ターミナルが苦手な人向けの逃げ道。同じClaude Codeを、黒い画面ではなくアプリの画面から使えます（後述）。

💡 はじめる前の大前提

Claude Codeを使うには、有料プランが1つ必要です。無料（Free）プランでは使えません。まず有料プランを契約するのが、すべてのスタートになります。

失敗しないための、たった2つのルール

RULE 01

入口は「公式」だけを使う

インストールも契約も、下の手順にある公式サイト・公式コマンドからだけ。偽サイトや広告リンクは踏まないこと。公式ページはブックマークしておけば、次から安全・確実です。

RULE 02

ブラウザはGoogle Chromeを既定に

ログイン認証はブラウザで行います。Safariでうまくいかないことがあるため、Chromeを「既定のブラウザ」にしておくと認証がスッと通ります。Chromeはこちらから無料で入れられます。

Mac × Chromeで、0からClaude Codeを導入する

パソコンが得意でない方でも、今日から触り始められるように作りました。5つのSTEPを上から順にどうぞ。所要時間は15分ほどです。

STEP1
CHECK YOUR MAC
Macの準備を確認する

まず、あなたのMacがClaude Codeを動かせるか、ざっくり確認します。ほとんどの方はそのままでOKです。

  • macOS 13.0（Ventura）以降であること。画面左上の🍎 →「このMacについて」で確認できます。古い場合は先にアップデートを。
  • メモリ4GB以上・インターネット接続があること（近年のMacならほぼ問題なし）。
  • 「ターミナル」という黒い画面のアプリを使います。場所は アプリケーション → ユーティリティ → ターミナル。⌘＋スペース で検索窓を出し「ターミナル」と打っても開けます。
🐣
黒い画面が怖い方へSTEP3で紹介するClaude Desktopアプリを使えば、ターミナルを開かずに“画面操作”だけで同じClaude Codeが使えます。まずは最後まで読んで、やりやすい方を選んでOKです。
STEP2
GET A PLAN
有料プランを1つ契約する

前提のとおり、Claude Codeは有料プランが必要です。Google Chromeで公式サイトを開いて、まずはProを契約しましょう。

  1. Chromeで claude.ai を開き、「Continue with Google」でGmailアカウントでログイン（初回は「18歳以上」の確認・規約同意）
  2. プラン画面で Pro を選ぶ（迷ったらProでOK・月約3,000円）
  3. クレジットカードを登録して契約完了 🎉
Claudeを開いて契約する ↗
うごく君のひとこと2回目以降も必ず同じ「Googleで続ける」から入ろう。メール＋パスワードで入ると別アカウントになり、契約や履歴が見えなくなるよ（よくある勘違い）。
STEP3
INSTALL
Claude Codeをインストールする

いよいよ本体を入れます。ターミナルに1行を貼り付けるだけ。Anthropic公式が推奨する、いちばん簡単な方法です。

  1. ターミナルを開く（STEP1参照）
  2. 下のコマンドをまるごとコピーして貼り付け、Enter
ターミナル — インストール（推奨）
$ curl -fsSL https://claude.ai/install.sh | bash
ちゃんと入ったか確認する

次のコマンドでバージョンが表示されればOK 🎉

ターミナル — 確認
$ claude --version
2.1.xx (Claude Code)
🛠️ うまくいかない時は
claude doctor と打つと、どこでつまずいているか自動で診断してくれます。command not found: claude と出たら、ターミナルを一度閉じて開き直すと直ることが多いです。
🧩
ターミナルが苦手な人の近道黒い画面を使いたくなければ、Claude Desktopアプリを入れて中の「Code」タブを使う方法も（中身は同じ）。claude.com/download からMac版をダウンロードするだけ。まずはこちらでも大丈夫です。
STEP4
LOG IN
Claude Codeにログインする

最後に、あなたの契約アカウントと結びつけます。ここでChromeが活躍します。

  1. ターミナルで claude と打って Enter
  2. ログイン方法を聞かれたら、サブスク（Pro/Maxなど）を選ぶ
  3. ブラウザ（Chrome）が自動で開くので、STEP2と同じアカウントで承認
  4. ターミナルに戻ると、ログイン完了。以降は聞かれません 🎉
うごく君のひとことブラウザが自動で開かない時は、ターミナルに出る長いURLを ⌘ を押しながらクリック、またはコピーしてChromeのアドレス欄に貼ればOKだよ。
STEP5
FIRST SESSION
はじめての「お願い」をしてみる

準備完了！ 作業したいフォルダに移動して claude と打つと、会話が始まります。まずは気軽に話しかけてみましょう。

仕事にも、遊びにも。任せられること

「コードを書く道具」と思われがちですが、本質は“ファイルを扱う作業ぜんぶ”の助手。プログラミングを知らなくても、日本語で頼めます。

💼 仕事で
  • 資料・報告書づくり：メモを渡して「読みやすい報告書に整えて」
  • 表計算・データ整理：CSVやExcelを渡して「重複を消して店舗別に集計して」
  • 議事録・文字起こし整形：長い記録を「決定事項・宿題つきで整えて」
  • マニュアル・FAQ作成：手順メモから清書された文書を自動生成
  • フォルダ整理・一括リネーム：散らかったファイルを規則的に並べ替え
  • 簡単な社内ツール・LP作り：「予約フォームのページを作って」も動く
🎮 遊び・暮らしで
  • 家計簿・献立アプリ：「今月の支出をグラフにするページを作って」
  • ちょっとしたゲーム作り：「じゃんけんゲームを作って」で完成品が動く
  • 写真・ファイルの一括整理：日付ごとにフォルダ分け、名前を揃える
  • 趣味の記録サイト：読書メモ・旅行記録を見やすいページに
  • 学びの相棒：「このコードが何をしているか初心者向けに説明して」
  • “作ってみたい”の入口：思いつきを、その場で形にして試せる

ここに“AIが加わる”と、何が変わる？

Claude Codeのすごさは、単発の作業ではなく「繰り返す作業を丸ごと任せられる」ところ。あなたの仕事のやり方そのものが変わります。

🔁毎日の定型作業が“ゼロ手間”に

  • 「毎朝この集計をして、この形式でまとめる」を一度教えれば、以降は一声かけるだけ
  • 人がやると30分の作業が、数十秒に短縮

🗂️散らかった情報が“資産”に変わる

  • バラバラのメモ・議事録・売上データを読み込ませて横断的に整理
  • 「先月と比べてどう？」にすぐ答えられる状態へ

🚀“作れなかったもの”が作れる

  • 未経験でも、社内ツール・LP・自動集計の仕組みが自分の手で
  • 外注や高価なソフトの前に、まず試せる

🧠あなたの“やり方”を覚えていく

  • /init で作るお約束メモに好みを書けば、毎回同じ品質で
  • 使うほど、あなた専用の助手に育つ
現場のイメージ飲食・建設・小売…どんな業種でも、「日々のExcel・書類・報告」をAIに任せて、人は判断と接客に集中する——これがClaude Codeで起きる一番大きな変化。0→1で触れば、自分の仕事のどこを任せられるかが見えてくるよ。

うごく君の「効くお願いの型」

Claude Codeへの頼み方は、次の4つを足すだけで結果が一段よくなります。ふわっと頼まず、具体的に。

1
何を例：この売上メモを
2
どうする例：報告書にまとめる
3
どんな形で例：A4・1枚・箇条書き
4
条件・注意例：専門用語はやさしく
▸ そのまま使えるテンプレ
〔このファイル／このフォルダ〕を読んで、
〔してほしいこと〕をしてください。
形式は 〔箇条書き5つ／表／1ページ〕で、
〔専門用語はやさしく／数字は正確に〕でお願いします。
うごく君のひとこと答えがイマイチでも大丈夫。「もっと短く」「この部分だけ直して」と会話でどんどん修正できるのがAIのいいところ。うまくいかない時は /clear でリセットして、頼み方を具体的にし直すのが一番の近道だよ。

安全に、かしこく使うための注意点

1大事な作業の前は「バックアップ」を

AIがファイルを書き換えることも。大切なフォルダは作業前にコピーかgit（バージョン管理）で保存点を作れば、いつでも元に戻せて安心です。

2「確認画面」は面倒がらずに読む

変更前に「これで良い？」と聞いてくれます。中身を見てから承認する習慣を。慣れるまでは“全部おまかせ”にしないのがコツ。

3機密情報は入れない

お客様の個人情報・カード番号・社外秘などは、基本的に入力しないのが安全。会社で使うなら社内ルールも確認を。

4答えを鵜呑みにしない

AIはもっともらしく間違えることも（ハルシネーション）。数字・名前・法律・金額は必ず自分で確認。最終判断は人が。

5“使いすぎ”の上限を知っておく

有料プランには「5時間ごと」「1週間ごと」の使用量上限が。話題が変わったら /clear でこまめにリセットすると、ムダな消費を防げます。

6最後は人が決める

Claude Codeは下書きと作業の助手。判断と最終チェックは自分で。公式URLはブックマークしておくと安心です。

よくある質問

プログラミングの知識がなくても使えますか？
はい。日本語で「〇〇して」と頼むだけです。コードを書けなくても、資料づくり・表の整理・ファイル整頓など“作業の助手”として十分に使えます。むしろコードが読めない人ほど「説明して」と気軽に頼めるのが強みです。
本当に無料では使えないの？
Claude Code自体は有料プラン（Pro以上）が必須で、無料プランには含まれていません。ただしブラウザ版のClaudeは無料で試せるので、「AIとの会話」に慣れる練習はそちらで。まずブラウザで慣れて、良さそうならProを契約——が失敗しない順番です。
WindowsのPCしか持っていません。どうすれば？
この記事はMac向けですが、Claude CodeはWindowsでも使えます。手順が少し異なるので、最新のやり方は公式のセットアップ手順を参照してください。ターミナルが苦手なら、Windows版のDesktopアプリから始めるのも手です。
ターミナル（黒い画面）がどうしても苦手です。
Claude Desktopアプリを使いましょう。claude.com/download からMac版を入れると、アプリの「Code」タブで画面操作のまま同じClaude Codeが使えます。この記事の「お願いの型」や活用アイデアはそのまま役立ちます。
Proで足りなくなったら、どうなりますか？
上限に達すると「しばらく待ってから」と表示され、一定時間で回復します。頻繁に当たるようになったらMax 5x（月約15,000円）への移行を検討する合図。プランを上げても履歴・設定はそのまま引き継がれます。
料金や仕様が、記事と違う気がします。
Claudeの料金・モデル・仕様は更新が速い分野です。この記事は2026年7月時点の情報ですが、最新の正確な内容は必ず公式の料金ページと公式ドキュメントでご確認ください。

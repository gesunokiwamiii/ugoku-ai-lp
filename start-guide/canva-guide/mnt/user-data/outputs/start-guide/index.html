START GUIDE

Googleアカウント1つで始める、
YouTube・ChatGPT・Claude

パソコンやスマホが得意でない方でも、今日から触り始められるように作りました。

💡 最初にこれだけ

Googleアカウント（Gmail）が1つあれば、YouTubeも・ChatGPTも・Claudeも、すべて同じアカウントでログインできます。まずGoogleアカウントを用意することが、すべてのスタートです。

失敗しないための、たった2つのルール

RULE 01

毎回「Googleで続ける」で入る

最初に Continue with Google で登録したら、2回目以降もずっと同じボタン。方法を混ぜると別アカウントができて、履歴が消えたように見えます（ChatGPTでよくある勘違いです）。

RULE 02

本物の公式サイトだけ使う

そっくりの偽サイトや広告に注意。下のボタンから開き、ブックマークしておけば、次からは安全・確実です。

STEP1
GOOGLE ACCOUNT
Googleアカウントを用意する

Googleが無料で配っている「あなた専用の鍵」。〇〇@gmail.com をすでにお持ちなら、もう完了しています（次のSTEPへ）。無い方は5分で作れます。

Googleアカウントを作る ↗

スマホ・パソコンどちらも同じ手順です。

  1. 作成ページを開く（accounts.google.com/signup）
  2. 姓・名 → 生年月日・性別を入力
  3. 希望のメールアドレスを決める（例：yamada.taro）
  4. パスワードを決める（8文字以上／英字・数字・記号）→ 必ずメモ
  5. 電話番号にSMSで届く6桁コードで本人確認
  6. 規約に同意して完成 🎉
うごく君のひとことメールアドレスとパスワードは、これから何度も使う“合鍵”。スマホのスクショではなく、紙のメモに残すのが一番安全だよ。

YouTubeにログインして使う

無料の動画サービス。ログインすると「チャンネル登録・履歴・コメント」が使えて便利になります。

YouTube を開く ↗

スマホは「YouTube」アプリが便利。

  1. 右上の「ログイン」をタップ
  2. 用意したGoogleアカウントを選ぶ
  3. 右上がアイコンに変われば成功
基本の使い方
  • さがす：検索窓にキーワードを入力
  • 登録：気に入ったチャンネルの「登録」＋🔔で新着通知
  • 保存：「後で見る」「再生リスト」で見たい動画をストック
こんな活かし方ができる
仕事 学び・集客に

  • 同業の人気動画を見て、売れ方・見せ方を研究する
  • スタッフ教育用に、手本動画を集めた再生リストを作る
  • 自社チャンネルで、お店・会社のファンを増やす
プライベート 暮らし・趣味に
  • 今日の献立やレシピを動画で見ながら作る
  • DIY・掃除・修理など「やってみたい」を手順で学ぶ

うごく君の「効くお願いの型」

AIへのお願い（プロンプト）は、次の4つを足すだけで結果が一段よくなります。

1
だれに向けて？例：取引先／お客様／初心者
2
何をしてほしい？例：メールを書く／要約する
3
どんな形で？例：箇条書き5つ／表／200字
4
どんなトーン？例：丁寧に／やわらかく／簡潔に
✕ もったいない例
「メールを書いて」
→ ふわっとした、使いにくい文章になりがち
◎ 効く例
「取引先に、納期変更をお詫びする丁寧なメールを、150字で書いて」
→ そのまま使える文章に近づく

ログインできない・困ったとき

登録した方法と同じ入口から入っていますか？
最も多い原因。Googleで登録したなら次もGoogleで。方法を混ぜると、空の別アカウントに入ってしまいます。
ブラウザを変える／キャッシュを消す／シークレットで試す
画面が進まないときに有効。別ブラウザ（Chrome・Safari）やプライベートウィンドウも試しましょう。
VPNをオフにする／認証メールは迷惑メールも確認
VPNでログインが止まることがあります。認証メールは迷惑フォルダに入りがちです。
それでもダメなときは、少し時間をおく
サービス側が一時的に混み合っている・障害中の可能性も。時間をおいて再度お試しを。

安全に、かしこく使うために

1パスワードを大切に

紙にメモして保管。他サービスと同じ使い回しは避けましょう。可能ならGoogleの2段階認証を。

2機密情報は入れない

お客様の個人情報・カード番号・社外秘などは、AIに入力しないのが基本です。

3答えは鵜呑みにしない

AIはもっともらしく間違えることも（ハルシネーション）。数字・名前・法律は必ず自分で確認を。

4最後は人が決める

AIは下書きと相談相手。判断と最終チェックは自分で。公式URLはブックマークしておくと安心です。

よくある質問

無料でどこまで使えますか？
3つとも無料で始められ、日常的な使い方なら無料でも十分です。ChatGPT・Claudeには回数などの上限があり、たくさん使う方は月額の有料プラン（各20ドル前後）で快適になります。
スマホだけでも大丈夫？
大丈夫です。3つとも公式アプリがあり、同じGoogleアカウントでログインすれば、パソコンとも履歴が同期します。まずはスマホから気軽にどうぞ。
ChatGPTとClaude、どっちから始めれば？
迷ったらChatGPTから。画像づくりやWeb検索も含めて何でも試せます。「長い文章を整えたい・要約したい」場面が多い方は、あわせてClaudeも。両方無料で持っておくのがおすすめです。
入力した内容は漏れませんか？学習に使われる？
学習利用の扱いはサービスや設定・プランで変わります。気になる場合は各サービスの設定（データ管理）を確認し、迷ったら機密情報は入れないのが安全です。会社で使うなら社内ルールも合わせて確認を。
英語が苦手でも使えますか？
問題ありません。3つとも日本語対応で、日本語で話しかければ日本語で返ってきます。ボタン表記が英語でも、この記事のとおり進めれば迷いません。
